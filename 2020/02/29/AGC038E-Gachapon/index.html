Description

з”ұдёҖдёӘйҡҸжңәж•°з”ҹжҲҗеҷЁпјҢжҜҸж¬Ўз”ҹжҲҗдёҖдёӘзҡ„ж•°пјҢжҜҸдёӘж•°еҮәзҺ°зҡ„жҰӮзҺҮдёәпјҲпјүгҖӮ

жҲ‘д»¬зҺ°еңЁиҰҒжұӮжҜҸдёӘж•°йғҪж»Ўи¶іе…¶еҮәзҺ°ж¬Ўж•°зҡ„жңҹжңӣж—¶й—ҙгҖӮ

е…¶дёӯгҖӮ

Solution

е…¶е®һпјҢжң¬иҙЁдёҠе°ұжҳҜжұӮдёҖдёӘйӣҶеҗҲпјҢе…¶дёӯжҜҸдёӘе…ғзҙ еҰӮжһңеӯҳеңЁпјҢиҜҙжҳҺе®ғе·Із»ҸеҮәзҺ°дәҶж¬ЎпјҢй—®иҝҷдёӘйӣҶеҗҲиў«еЎ«ж»Ўзҡ„жңҖжҷҡж—¶й—ҙзҡ„жңҹжңӣгҖӮ

еҫҲжҳҫз„¶пјҢиҝҷжҳҜе®№ж–Ҙзҡ„жЁЎеһӢпјҢжүҖд»ҘжҲ‘д»¬е…ҲеҘ—е®№ж–Ҙ

е…¶дёӯзҡ„ж„ҸжҖқжҳҜиҝҷдёӘйӣҶеҗҲдёӯзҡ„е…ғзҙ ејҖе§ӢеӯҳеңЁжҹҗдёӘеҮәзҺ°ж¬Ўзҡ„жңҹжңӣж—¶й—ҙгҖӮ

иҝҷдёӘй—®йўҳжҳҫз„¶жҜ”дёҖејҖе§ӢиҰҒжұӮзҡ„иҰҒеҘҪдёҖдәӣпјҢз„¶иҖҢиҝҳжҳҜж— жі•з®—гҖӮ

дёӢйқўжҳҜеҫҲйҮҚиҰҒзҡ„дёҖжӯҘиҪ¬еҢ–пјҢ

еҜ№дәҺдёҖдёӘйӣҶеҗҲпјҢеҶҚиҖғиҷ‘дёҖдәӣйӣҶеҗҲпјҢиЎЁзӨәеҮәзҺ°зҡ„ж¬Ўж•°гҖӮ

е®һйҷ…дёҠпјҢж №жҚ®жңҹжңӣзҡ„зәҝжҖ§пјҢзӯүдәҺиҝҷдәӣдёӯжҜҸдёӘйӣҶеҗҲиө°еҗ‘дёӢдёҖдёӘзҠ¶жҖҒзҡ„жңҹжңӣжӯҘж•°д№ҳд»ҘеҲ°иҫҫиҝҷдёӘзҠ¶жҖҒзҡ„жҰӮзҺҮгҖӮ

дёәд»Җд№Ҳпјҹ

жҲ‘д»¬иҖғиҷ‘жүҖжңүзҡ„дёӯзҡ„йӣҶеҗҲжһ„жҲҗдёҖеј жңүеҗ‘ж— зҺҜеӣҫпјҢжҜҸдёӘзӮ№иЎЁзӨәдёҖдёӘзҠ¶жҖҒпјҢиҝһеҗ‘жҜҸдёӘеҗҺз»§зҠ¶жҖҒдёҖжқЎиҫ№пјҢиҫ№жқғжҳҜиө°еҲ°йӮЈдёӘзҠ¶жҖҒзҡ„жңҹжңӣжӯҘж•°гҖӮжіЁж„ҸпјҢжҜҸдёӘиҠӮзӮ№еҸӘдёҺдёӯзҡ„е…ғзҙ зҡ„зӣёе…ігҖӮ

жҲ‘д»¬з»ҷжүҖжңүзҡ„иҠӮзӮ№иҝһеҗ‘дё”еҸӘиҝһеҗ‘дёҖдёӘз»ҲжӯўиҠӮзӮ№пјҢиҫ№жқғдёәпјҢи®ҫеҲқе§ӢиҠӮзӮ№гҖӮ

йӮЈд№Ҳе®һйҷ…дёҠпјҢе°ұзӯүдәҺеҲ°зҡ„жңҹжңӣжӯҘж•°гҖӮ

жҲ‘д»¬жғіжҠҠж•ҙдёӘиҙЎзҢ®жӢҶеҲҶжҲҗиҫ№зҡ„иҙЎзҢ®гҖӮ

жҲ‘д»¬иҖғиҷ‘еңЁиҝҷдёӘеӣҫдёҠпјҢиЎЁзӨәеҲ°иҫҫиҝҷдёӘзҠ¶жҖҒзӮ№зҡ„жңҹжңӣжӯҘж•°пјҢи®ҫдёәзҡ„дёҖдёӘеҗҺз»§зҠ¶жҖҒпјҢйӮЈд№Ҳжҳҫз„¶жңү

е…¶дёӯжҢҮзҡ„жңҹжңӣжӯҘж•°пјҢжҢҮзҡ„жҳҜд»Һзҡ„жҰӮзҺҮпјҢиЎЁзӨәеҲ°зҡ„жҰӮзҺҮгҖӮ

жҲ‘д»¬дёҚйңҖиҰҒжұӮеҲ°иҫҫжҜҸдёӘзӮ№зҡ„жңҹжңӣпјҢеҸӘйңҖиҰҒжұӮйӮЈдёӘжҖ»иҙЎзҢ®гҖӮ

жҲ‘д»¬и§ӮеҜҹжҜҸжқЎиҫ№зҡ„иҙЎзҢ®пјҢеҸ‘зҺ°е®һйҷ…дёҠе®ғзҡ„иҙЎзҢ®жҳҜ

еҜ№дәҺд»»дҪ•дёҖдёӘиҠӮзӮ№пјҢе®ғжңҖз»ҲдёҖе®ҡиғҪиө°еҲ°пјҢжүҖд»ҘпјҢйӮЈд№ҲжҜҸжқЎиҫ№зҡ„иҙЎзҢ®жҳҜ

жіЁж„ҸеҲ°еҜ№дәҺд»»дҪ•дёҖдёӘпјҢжҲ‘д»¬и®ҫ

е®һйҷ…дёҠпјҢжҳҜжҲ‘д»¬еҸҜд»Ҙз®—еҮәжқҘзҡ„пјҢеҚіжҜҸдёӘзҠ¶жҖҒеҲ°дёӢдёҖдёӘзҠ¶жҖҒпјҲдёҚз®Ўе…·дҪ“жҳҜд»Җд№ҲпјҢеҸӘиҰҒдёҺдёҚеҗҢпјүзҡ„жңҹжңӣжӯҘж•°пјҢе…·дҪ“жҖҺд№Ҳз®—пјҢжҲ‘д»¬еҗҺйқўеҶҚиҜҙгҖӮ

йӮЈд№ҲжҲ‘д»¬еҸҜд»ҘиҖғиҷ‘и®Ўз®—жҜҸдёӘзӮ№иҝһеҮәеҺ»зҡ„иҫ№еҜ№зӯ”жЎҲзҡ„иҙЎзҢ®е’ҢгҖӮ

йӮЈд№ҲпјҢжүҖжңүзҡ„иҙЎзҢ®е°ұеҸҜд»ҘеҶҷжҲҗ

иҝҷж ·пјҢжҲ‘д»¬е°ұиҜҒжҳҺдәҶгҖӮе®һйҷ…дёҠпјҢиҝҷдёӘжҺЁжј”ж–№жі•еҸҜд»ҘжҺЁе№ҝеҲ°еҫҲеӨҡзӣёе…ізҡ„йўҳзӣ®дёӯгҖӮ

W=\frac{S_T}{S}1+(1-\frac{S_T}{S})(W+1)\\
W=\frac{S}{S_T}

\frac{X!}{\prod_{i\in T}x_i!}\prod_{i\in T}(\frac{a_i}{S_T})^{x_i}

\frac{S}{S_T}\frac{X!}{\prod_{i\in T}x_i!}\prod_{i\in T}(\frac{a_i}{S_T})^{x_i}\\
=SX!*(\frac{1}{S_T})^{X+1}\prod_{i\in T}\frac{a_i^{x_i}}{x_i!}

XS_TT$$иҝҷдёӘйӣҶеҗҲжҖ»дҪ“ж— е…ігҖӮ

жҺҘдёӢжқҘд№ҹеҚҒеҲҶе·§еҰҷпјҒ

йӮЈд№ҲпјҢжҲ‘д»¬иҖғиҷ‘пјҢи®ҫиЎЁзӨәжүҖжңүзҡ„ж»Ўи¶іпјҢзҡ„е®№ж–Ҙзі»ж•°д№Ӣе’ҢжҳҜеӨҡе°‘гҖӮ

жҲ‘д»¬жһҡдёҫжҜҸдёҖдҪҚйҖүжҲ–дёҚйҖүжқҘиҪ¬з§»пјҢеҰӮжһңдёҚйҖүпјҢе°ұз»§жүҝдёҠдёҖж¬Ўзҡ„зӯ”жЎҲпјҢеҰӮжһңйҖүдәҶпјҢеӣ дёәеӨҡеҠ дәҶдёҖдёӘе…ғзҙ пјҢжүҖд»Ҙе®№ж–Ҙзі»ж•°еҸ–еҸҚпјҢз„¶еҗҺиҪ¬з§»дёҖдёӢе°ұеҘҪдәҶпјҢз»ҶиҠӮи§Ғд»Јз ҒгҖӮ

жіЁж„ҸпјҢеҲқе§ӢзҠ¶жҖҒдёәпјҢеӣ дёәе®№ж–Ҙзҡ„зі»ж•°жҳҜгҖӮ
