[LOJ3120]зҸҚзҸ 

еҸ‘иЎЁдәҺ 2020-03-19 | еҲҶзұ»дәҺ LOJ | | йҳ…иҜ»ж¬Ўж•°:
еӯ—ж•°з»ҹи®Ў: 1.6k | йҳ…иҜ»ж—¶й•ҝ ≈ 8

Description

жңүдёҖдёӘ$n$дёӘж•°зҡ„еәҸеҲ—пјҢжҜҸдёӘж•°еҸ–еҖјеңЁ$[1,D]$д№Ӣй—ҙпјҢй—®жңүеӨҡе°‘з§ҚеҸҜиғҪзҡ„еәҸеҲ—дҪҝеҫ—зӣёеҗҢзҡ„ж•°жңүиҮіе°‘$m$еҜ№пјҲдёҖдёӘж•°еҸӘиғҪеҮәзҺ°еңЁдёҖеҜ№йҮҢйқўпјүгҖӮ

$D\le 1e5,n,m\le 1e9$

жң¬йўҳжқҘжәҗдәҺCTS2019гҖӮ

Solution

жҲ‘д»¬еҒҮи®ҫдёҚиҖғиҷ‘йЎәеәҸпјҢи®ҫз¬¬$c$з§Қж•°жңү$cnt[c]$дёӘпјҢйӮЈд№Ҳйўҳзӣ®зӣёеҪ“дәҺ

йӮЈд№Ҳе®һйҷ…дёҠжҲ‘д»¬иҰҒжұӮжңүеӨҡе°‘з§Қ$cnt[i]$зҡ„еҸ–еҖјдҪҝеҫ—$\sum cnt[i]\% 2\le n-2m$пјҢжҜҸз§Қж–№жЎҲиҙЎзҢ®дёә

пјҢжңҖеҗҺеҶҚд№ҳдёҠ$n!$е°ұжҳҜзӯ”жЎҲгҖӮ

иҝҷдёӘжҳҫз„¶жҳҜжҢҮж•°еһӢз”ҹжҲҗеҮҪж•°зҡ„жЁЎеһӢпјҢйӮЈд№ҲпјҢжҲ‘д»¬жһ„йҖ жҢҮж•°еһӢз”ҹжҲҗеҮҪж•°гҖӮ

еӣ дёәеҸӘжңү$cnt[i]$дёәеҘҮж•°ж—¶жүҚдёҺйҷҗеҲ¶жңүе…іпјҢжүҖд»ҘжҲ‘д»¬еҸҜд»ҘжҠҠеҲҶдёәдёӨз§ҚпјҡеҘҮж•°дёҺеҒ¶ж•°гҖӮ

и®ҫ$A(x)=x+\frac{x^3}{3!}+\frac{x^5}{5!}+вҖҰ=\frac{e^x-e^{-x}}{2}$пјҢ$B(x)=1+\frac{x^2}{2!}+\frac{x^4}{4!}+вҖҰ=\frac{e^x+e^{-x}}{2}$гҖӮ

йӮЈд№ҲжҲ‘д»¬иҰҒеҒҡзҡ„е°ұжҳҜеҜ№дәҺжҜҸдёҖдёӘ$k\le n-2m$пјҢе°Ҷ$\binom{D}{k}A^k(x)B^{D-k}(x)$з¬¬$n$йЎ№зҙҜеҠ пјҢд№ҳдёҠ$n!$еҚіеҸҜгҖӮ

дҪҶй—®йўҳжқҘдәҶпјҢеӣ дёә$A$е’Ң$B$еҪўејҸйғҪдёҚеҘҪзңӢпјҲйғҪжңүдёӨйЎ№пјүпјҢеҰӮжһңжҡҙеҠӣдәҢйЎ№ејҸе®ҡзҗҶеұ•ејҖж— д»ҺдёӢжүӢпјҲеҗ¬иҜҙд№ҹиғҪеҒҡпјҹпјүгҖӮ

ж №жҚ®еҘ—и·ҜпјҢеҰӮжһңе…¶дёӯдёҖдёӘеҸҳжҲҗ$e^{tx}$зҡ„еҪўејҸпјҢеҫҖдёӢжҺЁзҡ„ж—¶еҖҷжӣҙйЎәеҲ©гҖӮ

йӮЈд№ҲпјҢжҖҺд№ҲеҠһе‘ўпјҹеҰӮжһңжҲ‘д»¬жҠҠе…¶дёӯдёҖдёӘеҸҳжҲҗ$e^{tx}$зӣёеҪ“дәҺйӮЈдёҖйғЁеҲҶиҰҒйҡҸдҫҝйҖүпјҢжүҖд»ҘжҲ‘д»¬иҖғиҷ‘е®№ж–ҘгҖӮ

жҲ‘д»¬и®ҫ$f[k]$дёәжҲ‘д»¬й’Ұе®ҡ$k$дёӘйўңиүІдёәеҘҮж•°пјҢеү©дёӢзҡ„йҡҸдҫҝйҖүзҡ„ж–№жЎҲж•°пјҢи®ҫ$g[k]$дёәжҒ°еҘҪ$k$дёӘйўңиүІдёәеҘҮж•°зҡ„ж–№жЎҲж•°пјҢйӮЈд№ҲжҲ‘д»¬жңү

ж №жҚ®дәҢйЎ№ејҸеҸҚжј”пјҢжҲ‘д»¬жңү

иҝҷдёӘдёңиҘҝжҳҫз„¶еҸҜд»ҘеҚ·з§Ҝз®—пјҢиҜ·иҜ»иҖ…иҮӘиЎҢжҺЁеҜјпјҲе…¶е®һжҳҜжҲ‘дёҚжғіжҺЁдәҶпјүгҖӮ

йӮЈд№ҲпјҢжҲ‘д»¬зҡ„й—®йўҳеҸҳдёәеҰӮдҪ•жұӮ$f[k]$пјҹ

ж №жҚ®е®ҡд№үпјҢ

еҫҲжҳҫз„¶зҡ„еҚ·з§ҜеҪўејҸпјҢзӣҙжҺҘ$NTT$еҚіеҸҜгҖӮ

Code

жҖ»з»“

иҝҷжҳҜжҲ‘з¬¬дёҖж¬ЎзңҹжӯЈж„Ҹд№үдёҠз”Ё$EGF$и§ЈйўҳпјҢйҰ–е…ҲпјҢз”Ё$EGF$дёҖиҲ¬йғҪиҰҒиҪ¬еҢ–жҲҗ$e^{tx}$зҡ„еҪўејҸпјҢиҖҢдё”е°ҪйҮҸйҒҝе…ҚеӨҡдёӘзұ»дјј$\frac{e^x+e^{-x}}{2}$иӢҘе№Іж¬Ўж–№зӣёд№ҳиҝҷз§Қжғ…еҶөеҮәзҺ°пјҲиҰҒз”ЁдәҢйЎ№ејҸе®ҡзҗҶеұ•ејҖпјҢеӨӘйә»зғҰпјүпјҢе°ҪйҮҸеӨҡеҮәзҺ°$e^{tx}$пјҢиҝҷдёӘж—¶еҖҷжҲ‘д»¬е°ұйңҖиҰҒе®№ж–ҘгҖӮ

е®№ж–Ҙзҡ„ж—¶еҖҷпјҢд»ҘеүҚз»ҸеёёжӢ…еҝғйңҖдёҚйңҖиҰҒеҠ з»„еҗҲж•°пјҢе®һйҷ…дёҠпјҢиҰҒеҠ з»„еҗҲж•°дёҖиҲ¬йғҪжҳҜз”ЁдәҢйЎ№ејҸеҸҚжј”пјҢжүӢжҺЁдёҖдёӢејҸеӯҗеҚіеҸҜгҖӮдёҖиҲ¬жғ…еҶөдёӢпјҢе®№ж–Ҙзҡ„жҳҜеӯҗйӣҶзҡ„ж—¶еҖҷдёҚз”ЁеҠ з»„еҗҲж•°пјҢдҪҶе®№ж–Ҙзҡ„жҳҜи¶…йӣҶзҡ„ж—¶еҖҷдёҖиҲ¬йғҪиҰҒеҠ з»„еҗҲж•°пјҢеӣ дёәи¶…йӣҶзҡ„иҙЎзҢ®дјҡжңүеӨҡж¬ЎпјҢиҖҢеӯҗйӣҶзҡ„иҙЎзҢ®дёҖиҲ¬еҸӘдјҡжңүдёҖж¬ЎгҖӮжҜҸж¬ЎеҒҡйўҳж—¶пјҢе…ҲжҺЁеҮәзұ»дјј$f[k]=\sum g[i]$зҡ„еҪўејҸеҶҚиҪ¬иҝҮжқҘпјҢдјҡеҮҸе°‘еҫҲеӨҡйЎҫиҷ‘гҖӮ

еҸҰеӨ–пјҢеҪ“з”ҹжҲҗеҮҪж•°з”ЁдәҺжҺЁејҸеӯҗзҡ„ж—¶еҖҷз”ЁеҲ°е…¶дёӯжҹҗдёҖйЎ№еҸҜд»Ҙз”Ё$[x^n]$жқҘиЎЁзӨәпјҢжӣҙдёәжғҠеҘҮзҡ„жҳҜиҝҷдёӘдёңиҘҝеҗҺйқўзҡ„дёҺз”ҹжҲҗеҮҪж•°ж— е…ізҡ„дёңиҘҝпјҲдёҺ$x$ж— е…ізҡ„дёңиҘҝпјүеҸҜд»ҘжҸҗеҲ°еүҚйқўжқҘпјҲе…¶е®һйҒ“зҗҶд№ҹеҫҲз®ҖеҚ•пјҢеҸӘжҳҜд№ӢеүҚе№¶жңӘжғіеҲ°пјүпјҢдҫӢеҰӮжң¬йўҳдёӯжҠҠ$\sum_{i=1}^k(-1)^{k-i}$жҸҗеҲ°еүҚйқўжқҘзҡ„жғ…еҶөпјҢдёҖе®ҡиҰҒзҶҹз»ғжҺҢжҸЎгҖӮ
