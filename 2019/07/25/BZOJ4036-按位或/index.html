Description

刚开始你有一个数字0，每一秒钟你会随机选择一个[0,2^n-1]的数字，与你手上的数字进行或操作。选择数字$i$的概率是$p[i]$。保证$0<=p[i]<=1$，$\sum p[i]=1$问期望多少秒后，你手上的数字变成$2^n-1$。

Solution
