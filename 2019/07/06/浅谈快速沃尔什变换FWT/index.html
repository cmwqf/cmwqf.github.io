еҝ«йҖҹжІғе°”д»ҖеҸҳжҚў

$FWT$дёҺ$FFT$зұ»дјјпјҢеҸӘдёҚиҝҮжҳҜиҝӣиЎҢйӣҶеҗҲеҚ·з§Ҝзҡ„и®Ўз®—пјҢеҰӮпјҡ

д№ӢеүҚзңӢеҲ°дёҖдёӘеҪўиұЎзҡ„жҜ”е–»пјҢжүҖи°“иҝҷдәӣеҸҳжҚўпјҢе°ұжҳҜзӣёеҪ“дәҺдҪ иҰҒиҝҮдёҖжқЎй©¬и·ҜпјҢдҪҶжҳҜзӣҙжҺҘиҝҮдёҚеҘҪиө°пјҢйӮЈд№ҲжҲ‘д»¬иҰҒдёҠдёҖдёӘеӨ©жЎҘпјҲжӯЈеҸҳжҚўпјүпјҢз„¶еҗҺд»ҺеӨ©жЎҘдёҠиө°иҝҮеҺ»пјҢжңҖеҗҺеҶҚиө°дёӢжқҘпјҲйҖҶеҸҳжҚўпјүгҖӮ

жҲ–(or)иҝҗз®—зҡ„FWT

жҲ‘д»¬иҖғиҷ‘жҲ–зҡ„$FWT$пјҢеҰӮжһңжҲ‘д»¬жңүдёҖдёӘйӣҶеҗҲ$A_i$пјҢи®ҫ$FWT(A)=AвҖҷ$йӮЈд№Ҳ$AвҖҷ_i$зҡ„ж„Ҹд№үе°ұжҳҜиҝҷдёӘйӣҶеҗҲжүҖжңүеӯҗйӣҶзҡ„иҙЎзҢ®д№Ӣе’ҢгҖӮ

иҝҷж ·пјҢеӣ дёә$k|(i|j)=k$еҸҜд»ҘжҺЁеҮә$k|i=k,k|j=k$жҲ‘д»¬еҸҜд»Ҙеҫ—еҲ°дёҖдёӘз»“и®әпјҡ

иҝҷж ·жҲ‘д»¬жқҘзңӢ$AвҖҷ$жҳҜжҖҺд№ҲжұӮзҡ„гҖӮ

жҲ‘д»¬иҖғиҷ‘еҗ§$A$й•ҝеәҰиЎҘдёә$2^k$пјҢеҲҶдёәдёӨеҚҠпјҢ$A0$жҳҜеүҚйқўдёҖеҚҠпјҢиЎЁзӨәз¬¬дёҖдҪҚжҳҜ0пјҢ$A1$жҳҜеҗҺйқўдёҖеҚҠпјҢиЎЁзӨәз¬¬дёҖдҪҚжҳҜ1гҖӮ

з„¶еҗҺпјҢжҲ‘д»¬д№ҹжҠҠ$AвҖҷ$еҲҶдёәдёӨеҚҠжқҘиҖғиҷ‘пјҢеүҚйқўдёҖеҚҠжҳҜз¬¬дёҖдҪҚжҳҜ0пјҢйӮЈд№Ҳе®ғзҡ„еӯҗйӣҶе°ұжҳҜ$A0вҖҷ$пјҲе®ғжң¬иә«пјүпјҢеҗҺйқўдёҖеҚҠз¬¬дёҖдҪҚжҳҜ1пјҢе®ғзҡ„еӯҗйӣҶж—ўжңү$A1вҖҷ$(е®ғжң¬иә«)пјҢиҝҳжңү$A0вҖҷ$гҖӮ

жүҖд»Ҙ

е…¶дёӯпјҢ$merge$е°ұжҳҜе°ҶдёӨдёӘж•°з»„жӢјжҺҘеңЁдёҖиө·пјҢ$+$иЎЁзӨәеҜ№еә”дёӢж Үзҡ„ж•°зӣёеҠ гҖӮ

иҝҷж ·жҲ‘д»¬е°ұиғҪеңЁ$O(nlogn)$зҡ„еӨҚжқӮеәҰеҶ…жұӮеҮәдәҶ$FWT(A)$гҖӮ

иҝҷж ·пјҢжҲ‘д»¬е°ұеҸҜд»Ҙз®—еҮә$FWT(C)$пјҢйӮЈд№ҲпјҢжҲ‘д»¬жҖҺд№ҲжҠҠ$FWT(C)$иҪ¬еӣһжқҘе‘ўпјҹ

е®һйҷ…дёҠпјҢеҶҷйӮЈд№ҲеӨҡејҸеӯҗд»ӨдәәзғҰиәҒпјҢиҖғиҷ‘еӣ дёәжҳҜ$or$иҝҗз®—пјҢжүҖд»ҘйҖ’еҪ’жҜҸдёҖдҪҚзҡ„ж—¶еҖҷпјҢеҸӘжңүеҪ“еүҚиҝҷдёҖдҪҚжҳҜ$0$зҡ„еҜ№еҗҺйқўзӣёеҜ№еә”зҡ„иҝҷдёҖдҪҚжҳҜ$1$зҡ„дә§з”ҹиҙЎзҢ®пјҢз„¶еҗҺйҖҶеҸҳжҚўе°ұжҳҜеҸҚиҝҮжқҘгҖӮ

дҪҶжҳҜиҝҷдёӘж—¶еҖҷе°ұдјҡжңүдёӘз–‘жғ‘пјҢжӯЈеҸҳжҚўжҳҜе…ҲеӨ„зҗҶе·ҰеҸідёӨиҫ№еҶҚз®—иҙЎзҢ®пјҢйӮЈд№ҲйҖҶеҸҳжҚўиҰҒжҠҠиҝҮзЁӢеҸҚиҝҮжқҘпјҢдёҚеә”иҜҘе…Ҳз®—иҙЎзҢ®еҶҚйҖ’еҪ’е·ҰеҸідёӨиҫ№еҗ—пјҢдёәд»Җд№ҲиҝҳжҳҜдёҖдёӘеҮҪж•°пјҹ

е®һйҷ…дёҠпјҢеӣ дёәиҖғиҷ‘жӯЈеҸҳжҚўж—¶жҜҸдёҖдҪҚзҡ„йЎәеәҸжҳҜж— жүҖи°“зҡ„пјҢжҲ‘д»¬жҢүдҪҚжӯЈзқҖз®—е’ҢеҖ’зқҖз®—жҳҜзӯүд»·зҡ„гҖӮ

жүҖд»ҘжҲ‘д»¬йҖҶеҸҳжҚўдҫҝеҒҮи®ҫжӯЈеҸҳжҚўжҳҜеҖ’зқҖжһҡдёҫжҜҸдёҖдҪҚжқҘи®Ўз®—зҡ„пјҢйӮЈд№Ҳе…ҲйҖ’еҪ’е·ҰеҸідёӨиҫ№еҶҚи®Ўз®—иҙЎзҢ®е°ұжҳҜеҗҲзҗҶзҡ„дәҶпјҢиҝҷдёӘиҰҒд»”з»ҶжҖқиҖғгҖӮ

з»“и®ә

дёӢйқўеҶҷ$IFWT$пјҢжіЁж„ҸпјҢжӯӨж—¶еӨ§жӢ¬еҸ·еҗҺйқўзҡ„$A$жҳҜе·Із»Ҹ$FWT$иҝҮеҗҺзҡ„$A$

Code

дёҺ(and)иҝҗз®—зҡ„FWT

иҜҒжҳҺеӨӘйә»зғҰдәҶпјҢзӣҙжҺҘдёҠз»“и®әеҗ§пјҢиҝҷдёӘе’ҢдёҠйқўжҲ–зҡ„е·®дёҚеӨҡпјҢдёҺзҡ„иҜқе°ұжҳҜжӯЈеҘҪеҸҚиҝҮжқҘпјҢи®©иҝҷдёҖдҪҚжҳҜ$1$зҡ„еҜ№иҝҷдёҖдҪҚжҳҜ$0$зҡ„иҝӣиЎҢиҙЎзҢ®гҖӮ

з»“и®ә

Code

ејӮжҲ–(xor)иҝҗз®—зҡ„FWT

иҝҷдёӘжҜ”иҫғйҡҫгҖӮ

дёҺеүҚйқўдёҖж ·зҡ„жғіжі•пјҢжҲ‘д»¬иҖғиҷ‘жһ„йҖ дёҖдёӘеәҸеҲ—пјҢдҪҶиҝҷдёӘеәҸеҲ—жҜ”иҫғеҘҮжҖӘгҖӮ

жҲ‘д»¬е®ҡд№ү$x\otimes y$дёә$x\& y$д№ӢеҗҺзҡ„дҪҚж•°зҡ„еҘҮеҒ¶жҖ§пјҲеҘҮж•°дёә$1$пјҢеҒ¶ж•°дёә$0$пјүгҖӮ

иҝҷдёӘиҝҗз®—ж»Ўи¶і$(i\otimes j)\oplus (i\otimes k)=i\otimes (j\oplus k)$пјҲжңүзӮ№еғҸд№ҳжі•еҲҶй…ҚеҫӢпјүпјҢеҸҜд»Ҙз”ЁжқҘиҝӣиЎҢејӮжҲ–еҚ·з§ҜгҖӮ

жҲ‘д»¬жһ„йҖ 

иҝҷдёӘзңӢиө·жқҘеҫҲеҘҮжҖӘпјҢдҪҶжҳҜ

йӮЈд№ҲпјҢиҝҷдёӘжҖҺд№Ҳ$FWT$е‘ўпјҹ

жӯЈеҸҳжҚўпјҢдҫқз„¶иҖғиҷ‘еҲҶжІ»пјҢеҲҶдёә$A_0$пјҢ$A_1$пјҢиЎЁзӨәеҪ“еүҚиҝҷдёҖдҪҚпјҲ$i$пјүжҳҜ$0/1$пјҢиҝҷдёӘж—¶еҖҷпјҢжҲ‘д»¬иҖғиҷ‘$j$е’Ң$j+2^i$пјҢпјҲ$j$зҡ„з¬¬$i$дҪҚдёә$0$пјүпјҢжҲ‘д»¬и®ҫ$[i,j]$иЎЁзӨә$i$иҝҷдёҖеұӮ$j$зҡ„еҖјпјҢйӮЈд№ҲжңүеҰӮдёӢеҮ з§ҚиҪ¬з§»пјҲиҖғиҷ‘дҪҚж•°дёӯ$1$зҡ„еҸҳеҢ–пјүпјҡ

$[i-1,j]->[i,j]$дҪҚж•°еўһеҠ зҡ„дёә$0\& 0=0$пјҢжүҖд»ҘиҙЎзҢ®дёә$+1$гҖӮ

$[i-1,j+2^i]->[i,j]$еўһеҠ зҡ„дёә$1\& 0=0$пјҢиҙЎзҢ®д№ҹдёә$+1$гҖӮ

$[i-1,j]->[i,j+2^i]$еўһеҠ зҡ„дёә$0\& 1=0$пјҢиҙЎзҢ®жҳҜ$+1$гҖӮ

$[i-1,j+2^i]->[i,j+2^i]$еўһеҠ зҡ„дёә$1\& 1=1$пјҢдҪҚж•°дёӯ$1$еҘҮеҒ¶жҖ§еҸ‘з”ҹеҸҳеҢ–пјҢз¬ҰеҸ·еҸ–еҸҚпјҢиҙЎзҢ®дёә$-1$гҖӮ
